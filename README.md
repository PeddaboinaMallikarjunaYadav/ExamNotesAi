# 🧠 ExamNotesAI

AI-powered notes platform built with MERN stack.

## 🚀 Features

* Generate structured notes using AI
* Visualize data using charts (Recharts)
* Generate diagrams using Mermaid
* Authentication system
* PDF export support
* Visualise previously Generated Notes

## 🚧 In Progress

* Performance optimizations
* Caching

## 🛠️ Tech Stack

* Frontend: React, Redux, Tailwind CSS
* Backend: Node.js, Express
* Database: MongoDB
* Visualization: Recharts, Mermaid

## ⚙️ Setup Instructions

### 1. Clone the repo

git clone (https://github.com/PeddaboinaMallikarjunaYadav/ExamNotesAi.git)

### 2. Install dependencies

cd client
npm install

cd ../server
npm install

### 3. Setup environment variables

Create `.env` files using `.env.example`

### 4. Run project

# backend

Go to the backend using: cd server <br />
npm run dev

# frontend

Go to the frontend using: cd client <br />
npm run dev
