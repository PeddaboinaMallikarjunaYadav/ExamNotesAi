# 🧠 ExamNotesAI

AI-powered notes platform built with MERN stack.

## 🚀 Features

* Generate structured notes using AI
* Visualize data using charts (Recharts)
* Generate diagrams using Mermaid
* Authentication system
* PDF export support

## 🚧 In Progress

* Stripe payment integration
* User history tracking
* Performance optimizations

## 🛠️ Tech Stack

* Frontend: React, Redux, Tailwind CSS
* Backend: Node.js, Express
* Database: MongoDB
* Visualization: Recharts, Mermaid

## ⚙️ Setup Instructions

### 1. Clone the repo

git clone <your-repo-url>

### 2. Install dependencies

cd client
npm install

cd ../server
npm install

### 3. Setup environment variables

Create `.env` files using `.env.example`

### 4. Run project

# backend

cd server
npm run dev

# frontend

cd client
npm run dev
